Zomato Data Analysis Project : 
This project performs an end-to-end Exploratory Data Analysis (EDA) on Zomato's restaurant dataset. By leveraging Python, we uncover actionable insights regarding consumer spending habits, restaurant ratings, and ordering preferences to help optimize business decision-making.

📊 Project Overview
As a data-driven professional at Zomato, the goal of this project is to analyze customer and restaurant patterns. We have processed the dataset to answer key business questions, including:

Which restaurant types are most popular?

How does online ordering impact restaurant ratings?

What is the "sweet spot" for pricing for a typical couple?

How do order modes (online/offline) vary across different restaurant types?

🛠️ Tech Stack
Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn

Environment: Google Colab

Platform: GitHub
